Commande fournisseur sur Divalto Guide pas à pas

Procédure complète pour créer une commande fournisseur dans Divalto, depuis la saisie de l'en-tête jusqu'à la duplication de l'article et la finalisation des lignes matériel/service.

📋 18 étapes 📷 10 captures d'écran ⏱ ~10 min
📑 Sommaire
① Création de la commande ② Gestion de l'article ③ Finalisation des lignes
①
Création de la commande fournisseur
Saisie de l'en-tête, sélection du fournisseur, adresses
1
Créer une nouvelle saisie de commande
Dans le menu Saisie des commandes, cliquez sur le bouton + pour démarrer une nouvelle commande.
2
Rechercher le fournisseur
Placez-vous dans le champ Fournisseur puis appuyez sur Ctrl + F pour ouvrir la recherche.
3
Saisir le code fournisseur
Tapez le code recherché — par exemple ECO.
4
Valider le fournisseur
Double-cliquez sur le fournisseur dans la liste pour le sélectionner et valider.
5
Renseigner le Dépôt
Champ Dépôt : saisir TTE puis VALIDER.
6
Référence interne
Champ Votre référence : saisir le N° de devis.
7
Renseigner l'en-tête
Onglet Entête : remplir le champ AFFAIRE au format T2533PRE.
8
Configurer les adresses
Onglet Adresses :
  • Livraison → Tiers adresse → Ctrl + F → taper saint benoit → sélectionner le bon site → double-clic
  • Code adresse → Ctrl + F → sélectionner le bon site → double-clic
②
Gestion de l'article (création par duplication)
Duplication d'une ligne existante, renommage, nettoyage de la sous-référence
9
Ouvrir la fiche Articles
Retournez sur la fenêtre DIVALTO principale puis cliquez sur Articles.
10
Dupliquer une ligne existante
Dans le champ Référence, saisir TMAT* pour filtrer.
Sélectionnez en bas une ligne existante du fournisseur, puis :
Fichier → Edition (en haut à droite) → Dupliquer → tout cocher.
11
Renommer la référence et la désignation
Validez la duplication, puis :
  • Modifier la Référence en TMAT2533 (où 2533 = n° d'affaire).
  • Si un second devis matériel existe : utiliser TMAT2533b.
  • Valider puis modifier la désignation.
💡
Convention de nommage : TMAT + n° affaire pour la part matériel, suffixe b, c… pour les commandes additionnelles.
12
Sélectionner la ligne créée
Sélectionnez la nouvelle ligne dans la liste pour pouvoir intervenir sur ses sous-références.
13
Accéder à la sous-référence
Cliquez sur l'onglet Sous-réf.
14
Supprimer la sous-référence
Cliquez sur le bouton X pour supprimer la sous-référence existante (héritée de la duplication).
15
Confirmer la suppression
Valider la suppression de la fiche puis Valider à nouveau.
16
Réinitialiser les valeurs
Cliquez ensuite sur le bouton +, mettez toutes les valeurs à 0, validez et fermez.
✓
Votre article est maintenant prêt à être utilisé dans la commande fournisseur.
③
Finalisation des lignes de commande
Retour sur la commande fournisseur — saisie des lignes matériel et service
17
Ajouter la ligne matériel
Retournez sur la COMMANDE FOURNISSEUR puis dans Ligne :
  • Cliquer sur +
  • Référence : TMAT2533
  • Vérifier la désignation
  • Qté : 1
  • Montant : saisir la part matérielle
  • VALIDER
18
Ajouter la ligne service
Répéter la même procédure pour la part service en utilisant la référence TSER… (sur le même principe que TMAT).
📝
La convention TSER + n° affaire permet de séparer matériel et prestation pour le suivi comptable.
🎉

Commande fournisseur créée !

Vous pouvez désormais imprimer, transmettre ou archiver votre commande dans Divalto. Utilisez le bouton « Imprimer le guide » en haut à droite pour conserver une version papier de cette procédure.
